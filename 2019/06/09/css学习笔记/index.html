CSS学习笔记

文本

文本颜色color

文本对齐text-align

文本修饰text-decoration

文本转换text-transform

指定特定文本转换为大写、小写或者单词的首字母大写

文本缩进text-indent

文本缩进用来指定文本的第一行的缩进,像素为单位，比如段落的开头

CSS字体

css字体属性定义字体类型、加粗、大小、文字样式等

字型font-family

  • 通用字体系列–拥有像似外观的字体系统组合（如 “Serif” 、”SansSerif”或 “Monospace”）
  • 特定子体系列–一个特定的字体系列（如”Times”或”Courier”）

字体样式font-style

主要用于指定斜体文字的字体样式属性

字体大小font-size

  • 按照像素定义字体大小

  • 按照em来设置字体大小,1em和当前浏览器默认的字体大小相等，一般是16px,但是在IE中可能兼容性不好

  • 使用百分比和EM组合，可以解决不同浏览器兼容问题，设置 <\body>元素的默认字体大小的是百分比：

字体粗细font-weight

字体转换font-variant

显示结果如下：
p.normal:
My name is Hege Refsnes.
p.small:
MY NAME IS HEGE REFSNES.
